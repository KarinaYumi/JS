Eventos em JS

Os eventos são basicamente um conjunto de ações que são realizadas em um determinado elemento da página web, seja ele um texto, uma imagem, ou uma div, por exemplo.

Por exemplo, se o usuário clica em um botão numa pagina web, você pode querer responder a esta ação mostrando na tela uma caixa de informações.

Alguns Eventos

OnClick Disparado quando clicado
dblClick Disparado quando clicado duas vezes
input Disparado sincronicamente quando o valor de um elemento input, select, ou textarea é alterado.
onMouseOut Disparado quando o usuário retira o mouse sobre o elemento
onMouseOver Disparado quando o usuário passa o mouse sobre o elemento
onFocus o elemento é focado
onKeyDown TECLADO -Dispatado quando a tecla é pressionada
onKeyUp TECLADO - Disparado quando a tecla é solta
onLoad Disparado quando qualquer elemento for carregado
onChange Dispara uma ação quando uma mudança ocorre em algum elemento.
